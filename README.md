# Asset Management System

The Asset Management System is a full-stack web application designed to help organizations manage, track, and monitor their assets efficiently. The system provides a centralized platform for handling asset allocation, maintenance, and lifecycle management.

This project is developed using Angular for the frontend, ASP.NET (C#) for the backend, and SQL Server for database management.

## 🚀 Features
- Asset creation and management
- Asset allocation and tracking
- User management and role-based access
- Asset history and status monitoring
- CRUD operations for assets, users, and categories
- Secure API integration between frontend and backend

## 🛠️ Technologies Used
- **Frontend:** Angular
- **Backend:** ASP.NET Web API (C#)
- **Database:** SQL Server
- **Architecture:** RESTful API

## 📌 Use Case
This system is suitable for organizations looking to digitally manage physical or IT assets such as laptops, equipment, or inventory, ensuring better tracking, accountability, and maintenance.

## 🎯 Objective
The main objective of this project is to provide an easy-to-use and scalable solution for asset management while following clean architecture and industry best practices.
